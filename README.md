# Math Challenge

Это образовательный тренажер по математике, оформленный в стиле современной игры

## 🎮 Возможности
- Тренировка умножения и деления.
- Полная таблица умножения (без дубликатов).
- Режим "ВСЕ" (смешивание умножения и деления).
- Система "Страйков" (серия дней ежедневных занятий).
- Повтор ошибок: если ответили неверно, пример попадет в список для повторения.
- Красивый неоновый дизайн, звуковые эффекты и статистика!
